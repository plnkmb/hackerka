# Docker welcome task

## Что делает проект

Этот проект собирает Docker-образ на базе Ubuntu, копирует внутрь файл `conf/welcome.txt` и при запуске контейнера выводит содержимое файла. После этого контейнер остаётся запущенным.

## Состав проекта

- `Dockerfile` — инструкция по сборке Docker-образа
- `conf/welcome.txt` — приветственный файл, который копируется в образ
- `README.md` — описание проекта и команд

## 10 команд Linux

1. `pwd` — показать текущую директорию
2. `ls` — показать содержимое директории
3. `cd` — перейти в другую директорию
4. `mkdir` — создать директорию
5. `touch` — создать пустой файл
6. `cp` — скопировать файл или директорию
7. `mv` — переместить или переименовать файл
8. `rm` — удалить файл или директорию
9. `cat` — вывести содержимое файла
10. `head`, `tail`, `less` — просмотр файла

## 5 примеров команд с путями

- `pwd`
- `ls /`
- `mkdir /tmp/testdir`
- `touch /tmp/testdir/file.txt`
- `cat /welcome.txt`
