# Docker task: what is PID

## Что делает проект

В рамках этой задачи был создан отдельный Docker-образ в папке `docker/pidoras`.  
Образ собирается на базе `debian:12-slim` и содержит базовые Linux-команды, которые нужны по заданию: `chmod`, `chown`, `ps`, `top`, `kill`, `id`, `whoami`.

После сборки образ был:
- собран локально;
- перетегирован для Docker Hub;
- запушен в Docker Hub;
- использован для запуска контейнера уже из образа из registry.

Внутри контейнера были выполнены действия, которые требовались по заданию:
- создан файл;
- изменены права на файл;
- запущен процесс `sleep 1000` в фоне;
- найден PID процесса;
- процесс был завершён.
